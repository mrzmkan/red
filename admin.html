# ناوی کتێب مامۆستا بەروار (هیجری) جۆری ئیجازە کردارەکان

👤 زانیاری پرۆفایلی زانا

🔑 ڕێکخستنی GitHub

ناوی بەکارهێنەرت لەسەر GitHub (نمونە: ahmad-dev)

ناوی ڕیپۆزتری کە مالپەڕەکەت تێدایە

تۆکێنی دەستڕاگەیشتن دروستبکە لە GitHub → Settings → Developer settings → Personal Access Tokens → Tokens (classic)
پێویستە repo چێکباندرێت (خانەی repo تیک بکەوە). تۆکێنەکە تەنها یەکجار نیشاندەدرێت — کۆپیش بکە.

ℹ️ ئامۆژگاری دامەزراندن

١. دروستکردنی GitHub PAT:
بچۆ بۆ GitHub → Settings → Developer Settings → Personal Access Tokens → Generate New Token.
پاشان چێکی repo بکەوە و تۆکێن بکۆپی.

٢. ڕێکخستنی:
ناوی بەکارهێنەر، ناوی ڕیپۆزتری و تۆکێنت دابنێ لە سەرەوە، پاشان پاشەکەوتبکە.

٣. تاقیکردنەوە:
دوگمەی "تاقیکردنەوەی بەستەر" دابگرە بۆ دڵنیابوون لە دووستی GitHub.

٤. بەرێوەبردن:
ئیجازەکان زیادبکە، دەستکاری بکە یان بسڕەوە. هەموو گۆڕانکارییەک خۆکارانە بۆ فایلی data.json لە GitHub دەگات.
